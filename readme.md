
# Foto Postgre Downloader




## Installasi

Buat venv python

```bash
  py -m venv nama_venv
  .\nama_venv\Scripts\activate
```

Clone repository

```bash
  git clone url_repo
``` 

Install requirement

```bash
  pip install -r repo\requirements.txt
``` 

Project sudah siap dijalankan 

```bash
  cd repo
  py foto_single.py
```
